PageData

PageData class

Defines the Inertia page payload structure.

PageData is serialized to JSON and sent to the client during Inertia visits.

final page = PageData(
  component: 'Dashboard',
  props: {'user': {'name': 'Ada'}},
  url: '/dashboard',
);
