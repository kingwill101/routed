# Testing

Use `AssertableInertia` and the test extensions to verify page payloads
without manually inspecting JSON.

## Basic assertion

```dart
response.assertInertia((page) {
  page.component('Dashboard').has('user.name');
});
```

## What to test

- component names
- resolved props
- deferred and merge metadata
- flash and error payloads
- SSR and HTML bootstrap behavior when relevant

## Helpful pairings

The `HttpServer` helpers are small enough to exercise in unit or integration
tests directly:

- `buildInertiaHttpPageData()`
- `buildInertiaHttpResponse()`
- `respondWithInertiaPage()`
- `writeInertiaResponse()`
