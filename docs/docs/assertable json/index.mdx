# assertable_json

A powerful, fluent JSON testing utility for Dart that makes it easy to write expressive and readable assertions for JSON data structures.

## Overview

The `assertable_json` package provides a fluent API for testing JSON data in Dart applications. Inspired by Laravel's JSON testing utilities, it allows you to write clear, readable assertions for complex JSON structures.

## Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  assertable_json: ^0.1.1
```

Or install via command line:

```bash
dart pub add assertable_json
```

## Basic Usage

```dart
import 'package:assertable_json/assertable_json.dart';
import 'package:test/test.dart';

void main() {
  test('verify user data', () {
    final json = AssertableJson({
      'user': {
        'id': 123,
        'name': 'John Doe',
        'email': 'john@example.com',
        'age': 30,
        'roles': ['admin', 'user']
      }
    });

    json
      .has('user', (user) => user
        .has('id')
        .whereType<int>('id')
        .has('name')
        .whereType<String>('name')
        .has('email')
        .whereContains('email', '@')
        .has('age')
        .isGreaterThan('age', 18)
        .has('roles')
        .count('roles', 2));
  });
}
```
